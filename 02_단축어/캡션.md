# 02 릴스 — 클로드 단축어 5개 (캡션)

긴 프롬프트 매번 다시 치세요? 한 단어로 끝낼 수 있어요. 🔖 저장각

자주 쓰는 긴 프롬프트를 '단축어'(약속어)로 등록해두면, 누가 쳐도 「/보고」 한 단어로 같은 포맷이 나옵니다. 팀 프로젝트에 공유하면 결과물도 통일돼요.

바로 쓰는 5개 👇
/정리 — 핵심 3줄 + 실행 포인트
/보고 — 두괄식 한 장 보고서
/검토 — 약점·리스크 깐깐하게
/영어 — 비즈니스 영어로
/생각 — 아이디어 10개 + 난이도

※ '/이름'은 앱 기능이 아니라 내가 정한 약속어 — 프로젝트가 기억해서 따릅니다.
만드는 법은 3줄: 프로젝트 만들기 → 지침에 "'/정리'라고 하면 = (긴 프롬프트)" → 대화에서 /정리.

🎁 단축어 5종 세트 + 등록법(한글 별명 버전 포함) 통째로 드릴게요.
댓글에 "단축" 눌러주시면 → DM으로 바로 보내드립니다. (자동 발송)

저장 🔖 + 팔로우(@claude_lab.kr)하면 다음 편도 놓치지 않아요.
